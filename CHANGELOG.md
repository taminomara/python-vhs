# Changelog

## [unreleased]

## ## [1.1.1]

- Removed forgotten debug print.

## [1.1.0]

- Added CLI interface: `python -m vhs ...` now resolves and invokes VHS.
- Added an option to install VHS from custom GitHub repo (to use [SVG fork]).
- Releases are no longer uploaded to test version of PyPi.

## [1.0.0-post1]

- Moved documentation to Read the Docs and updated all links.

## [1.0.0]

- Initial release.

## [1.0.0-beta0]

- Pre-release.

- Added `max_version` parameter.

- Added `auth` parameter to pass GitHug auth token.

- Small bug fixes.

- Updated CI and dependencies.
